抗辐照加固的三种措施，即工艺加固、设计加固和封装加固。

1 工艺加固

​ 材料和工艺加固就是通过改进和创新材料和工艺来达到抗辐照效果，辐照主要是通过栅氧化层中的陷阱电荷和界面态电荷来影响 MOSFET 的正常工作，因此，我们需要采取措施来改进栅氧化层的性能。目前有以下三种常见的方法：高k栅介质、深亚微米 MOS 器件和 SOI 工艺。

2 设计加固

设计加固是指通过改变电路结构，来提高电路的抗辐照性能。它是目前为止最为普遍和广泛的抗辐照加固方法。设计加固的方式最大的优点是兼容商用工艺线，降低了工艺成本，在禁运限制严格的当下很适合国内发展。同时，针对不同的电路节点，设计人员可以通过判断其对辐照敏感性的不同采取不同的加固电路设计方式，以此来提高电路运行速度、降低版图面积。但是设计加固依旧会在很大程度上增加版图面积，也会使得电路延迟升高、运算速度下降。

目前，设计加固的方法多种多样，主要有检错纠错码技术、保护门技术、双向互锁存储单元技术和三模冗余等

2.1三模冗余技术

三模冗余（TMR Triple Modular Redundancy）其实是一种容错设计技术，简单地说就是将一份数据同时保存到三个存储空间，这三个存储空间如果有一个在辐照情况下发生了翻转，另外两个正确的数据还是可以输出正确的结果。

输入数据同时存入三个存储单元中，三个存储单元通过线路连接到一个表决器，表决器的作用就是选出三个存储单元的较多数的结果，输出到后级电路。例如在辐照的干扰下。有一个存储单元的数据发生翻转，另外两个没有翻转，则表决器就会选出那两个没有翻转的存储单元输出的数据作为整个电路的输出结果。

2.2检错纠错码技术

保护存储器数据的准确是抗辐照工作的主要内容，通过对存入存储器数据进行编码，在读出时进行解码，可以发现错误或者修改错误，从而达到抗辐照的目的。

纠错码一般常见的有循环冗余码（CRC）和汉明码（Hamming Code），其中循环冗余码用于多位串行数据传输中的检错纠错，而汉明码用于多位并行数据传输中的检错纠错，如果大量使用例如汉明码的纠错码，由于纠错功能需要的电路非常复杂，会占用过多的逻辑资源，同时也会减慢电路的速度。

存储器在受到辐照后，可能会发生单粒子翻转现象，造成某位存储数据的翻转，但多个存储单元同时翻转的概率还是很低的，所以能够纠正一位错误的汉明码在存储器中的应用比较常见。但随着集成电路尺寸的逐年减小，存储器的空间也被一直压缩，每个存储单元之间的间距也越来越小，入射的高能粒子可能一次造成多个存储单元的翻转，造成多位错误的发生。

保护门电路主要用于消除单粒子翻转的影响，组合逻辑产生输出结果，在 B 输入端的前面加一级延时单元，当组合逻辑受到辐照后，可能会发生单粒子翻转，造成输出结果相反，当组合逻辑的结果翻转的时间小于延时单元设定的时间时，输出 Y 就不会产生错误，当组合逻辑的结果翻转的时间大于延时单元设定的时间时，输出 Y 还是会产生错误，因此，我们需要合理设置延时单元的时间，以达到符合要求的抗辐照性能。

2.4双向互锁存储单元技术

基本 DICE 结构如图所示，图中可以看到，所有字母 P 的管子均为 PMOS 管构成的反相器，所有字母为 N 的管子均为 NMOS 管构成的反相器，节点 X0X3 是 4 个数据锁存节点，负责存储数据。基本的 DICE 结构拥有 4 个双稳态电路，他们分别由 P1N0、P2N1、P3N2 和 P0N3 构成，这就造成了四个存储节点 X0X3 存储的数据互反，四个节点的数据只会出现 0101 或 1010 的情况。CK 一般为时钟信号，它可以控制 N4N7 的开关，从而进行数据的读写操作，N4N7 这四个 NMOS 管在这里被当作传输管使用。

从图中可以看出，X0~X3 四个数据节点相互制约，X1 通过 N0 来影响 X0，同时 X0 通过 P1 来影响 X1，其他的节点原理类似，数据通过传输管存入存储节点后，在下一个数据来临之前会一直保持不变，当电路暴露在辐照环境下之后，某一个 NMOS 或 PMOS 发生了单粒子翻转效应，造成了某一存储节点的数据翻转，这个节点错误的传输会被限制，即4个存储节点不会全部出错，同时通过其他相邻的正确节点数据来修正错误的节点。

DICE 结构最关键的部分就是增加了存储节点的个数，由普通结构的2个节点变为了 4 个节点，通过双稳态结构控制存储节点的数据，将4个节点有限隔离， 这样当发生短暂辐照时，一个节点的错误不会影响所有节点，错误还会被其他正确的节点所修正，前提是这个错误不是长期发生的，例如X0节点数据发生翻转， 由‘0’变为‘1’，那么N3会导通，X3 会由‘1’变为‘0’，但 X2 和X0数据不会被影响，综上所述基于该结构的存储单元具有较好的抗单粒子翻转能力。

2.5 版图加固

电路的版图抗辐照加固是抗辐照设计加固的一个重要的领域，能够有效防止总剂量辐照的发生，由于现有工艺的不断改进，CMOS 器件中的栅氧化层厚度不断减少，总剂量效应对于 MOS 管阈值电压的影响越来越小，但是对于场氧化层的影响越来越严重，总剂量效应在场氧化层产生的漏电流会造成 MOS 器件之间的隔离失效，也会增大器件内部源漏之间的泄露电流，从而增大系统功耗。在设计中，我们常常通过改变MOS 管的栅的形状来达到抗总剂量辐照的作用， 一般来说有环形栅和半环形栅。

剂量辐照会使 MOS 管的源漏两端之间产生漏电流，由于栅极的氧化层非常的薄，氧化层陷阱电荷或界面态电荷会比较少，相当于电阻较大，通过栅极的漏电流就会变得很少，但是场氧化层比较厚，内部会存在大量的氧化层陷阱电荷，这样就会有很多的源漏之间的漏电流经过，造成源漏短路。

一些简单的抗辐照版图加固方法是通过加长加宽源漏之间的漏电流通道，从而使得漏电流减少，例如H型栅结构或锯齿形栅结构，通过加长或加宽源漏之间的栅氧化层的长度，来减少漏电流。

环形栅结构就是使用栅级氧化层将源级或漏级全部包围起来，通过栅氧化层隔离源级和漏级，这样就基本排除了漏电流通过场氧化层传播，大大减少了总剂量效应产生的泄露电流，可以有非常好的抗辐照效果。但这样也会占用比较大的面积，造成版图面积过大。半环栅和环栅的原理相同，但占用的面积比较小，且适用于较少宽长比的晶体管。

3封装加固

封装加固，封装为芯片提供保护、散热、电源连接等功能。目前，比较常用的封装材料有低温共烧陶瓷材料、高导热率氮化铝陶瓷材料、AlSiC 金属复合材料等。

屏蔽式的封装技术可以使芯片抗辐照能力提高 2-3 个数量级。能够非常有效的减轻空间及核辐照对集成电路芯片的损害，而且封装屏蔽成本低，见效快，操作简单，与之前介绍的工艺加固和设计加固相比，大大节省时间和金钱，是一种性价比很高的抗辐照加固方式。抗辐射封装加固技术的核心，是将集成电路芯片与空间辐射环境进行适当的屏蔽和物理隔离。封装加固是一种扩展 CMOS 器件使用寿命的直接而有效的途径。

封装加固可选用的材料非常丰富，美国很早就开始研究封装加固的原理和材料，现在已经发展出了塑料封装，重金属封装，金属陶瓷封装等封装技术。

辐照来源和辐照效应

发表于 2020-07-31 更新于 2020-08-04 分类于 Antirad
本文字数： 4.2k 阅读时长 ≈ 4 分钟

1. 辐照来源

空间辐射和人造辐射是两种最主要的辐射环境。其中，空间辐射环境可以分为范艾伦辐射带和宇宙射线两种，人造辐射环境可以分为核武器爆炸和地面辐射模拟源两种。

现代社会，宇宙中布满了各式各样的卫星设备，所有的卫星航天器全都离不开半导体器件。空间环境中存在着多种辐射，许多的电子设备就是在这些太空辐照的影响下出现故障，导致整个航天器的非正常工作。因此，针对抗辐照的研究是设计制造航天器延长航天器使用寿命的关键技术，同时也是现代电子研究的热门领域。

在太空环境中，存在着非常复杂的辐照环境，其中有 γ 射线、X 射线及高能重离子。其中高能重离子会进入器件材料中去，和其中的原子发生碰撞，从而将能量传递给这个原子，被碰撞的原子又会继续碰撞另外的原子，这样不断碰撞下去， 就会使器件材料内部产生大量的空隙原子对，使得材料的结构发生混乱，从而影响了半导体材料的性质，如少数载流子寿命、多数载流子浓度和载流子器件中的数据扰动。这种情况严重的会导致器件失效。

辐照效应是一个比较复杂的课题，它是一个综合性很强的学科，涵盖了微电子学、半导体理论、电路分析理论等学科。同时，辐照也是有着很多种类型的， 不同种类的辐照对器件的影响也不同，对半导体器件产生影响的主要辐照类型有四种：质子、中子、电子和 γ 射线。对于一般的半导体器件来说，辐照引发的电离效应和位移效应是造成器件性能退化甚至发生错误的最重要的两个效应。对微电子器件产生重要影响的且研究最多的因素主要有 γ 总剂量辐射、γ 剂量率辐射、中子辐射和单粒子效应。

在两个基本效应中，辐照引起的电离效应又可以分成三种类型来讨论：总剂量效应、剂量率效应和单粒子效应。

也可以分为位移效应、电离效应和表面效应。

2 电离辐照效应

器件经过一定能量的辐照后，在康普顿效应、光电效应及辐射粒子碰撞的作用下，会使半导体材料中的原子或分子激发出电子空穴对，形成了带负电的自由电子和带正电的失去电子的原子或分子。激发出的自由电子有些仍然具有很高的能量，它会继续与其他原子或者分子发生碰撞，产生新的电子空穴对，这些在材料中存在的电子空穴对引起了电离辐照效应。我们将它分为总剂量效应、剂量率效应和单粒子效应来讨论。

2.1 总剂量效应

工作在太空中或接触核辐射的微电子器件，往往会较早的失效或出现错误， 人们后来对其研究后发现，造成器件失效的原因，主要是由于微电子器件长期受到高强度的辐射，辐射的高能粒子或光子冲入器件内部，使得器件内部的正电荷产生堆积，严重影响器件的正常工作。因此，处在辐射环境中的微电子器件工作寿命会大大降低，如果没有做好完善的抗辐照措施，还会导致严重的航天事故， 造成不可挽回的损失。

总剂量效应一般体现在半导体元件的氧化层中，之所以叫总剂量效应，是因为器件所受的辐射越多，内部积累的电子和空穴对就越多，在氧化层中，电子的迁移率远大于空穴，因此电离出来的电子会被电场迅速赶出氧化层，而带正电的空穴会运动到界面处，在移动期间，一部分空穴会被氧化层中的陷阱捕获，形成固定正电荷，另一部分会被界面处的陷阱捕获，形成界面态。这两种是总剂量效应的主要作用机制，在后面的 2.1.3.3 会详细解释。

准确地来说，半导体器件在辐照曝光时间内由于电离效应而造成的半导体晶格累积损伤可以称之为总剂量效应。它以 rads 为单位进行测量，总剂量效应会导致器件的性能逐渐下降，最终造成器件的失效。在 CMOS 器件中，辐射在栅绝缘层中形成电子空穴对，复合过程中产生光电流，陷于绝缘体晶格缺陷中的空穴会产生持久的栅极偏置并影响晶体管的阈值电压。

总剂量效应对于半导体器件有多方面的影响，主要的有三种：一是 MOS 管沟道电子迁移率降低，严重影响器件性能，这是因为处于氧化层界面处的正电荷起到了主要作用，正电荷会对沟道中的导电电子产生吸引作用，产生库伦散射，使得载流子迁移率减少，这就会使得半导体器件的性能产生下降，需要更大的栅极电压才能恢复正常的性能；二是阈值电压漂移，由于界面和氧化层中存在了很多正电荷，所以对于 NMOS 来说会导致阈值电压降低；三是会造成 MOS 管的泄漏电流增加，主要是场氧化层的器件内部漏电流和器件之间的泄漏电流[11]。

2.2 剂量率效应

剂量率是指单位时间内接受的，称为剂量率，通常用于表示辐射场的强弱，单位为拉德/小时（rad/h），一般情况下，剂量率越大越显著，但当剂量率达到一定程度时，效应与剂量率之间失去比例关系。短时高强度辐射脉冲，通常发生在核爆炸期间。高强度粒子流在半导体器件内产生光电流，从而导致晶体管随机打开，改变触发器或存储单元的逻辑状态。如果脉冲持续时间太长， 则可能会造成永久性损坏。

2.3 单粒子效应

单粒子效应，简单的来说就是单个高能粒子入射到微电子器件中后，导致器件的工作状态发生改变，存储器的数据丢失，或者造成不可逆的永久性损坏。

单粒子效应是影响半导体器件的最为繁琐和严重的效应。同时，随着时代的发展科技的进步，卫星越来越多，非常多高端的微电子器件被应用于卫星上，针对半导体器件的抗辐照加固越来越重要。

1954 年至 1957 年，美国在地面核试验期间，电子监测设备中观察到许多异常情况时，首次发现了单粒子干扰现象。在 20 世纪 60 年代，空间电子器件中观察到了更多的问题，但当时很难将辐照导致的软故障与其他形式的错误区分开来。

1972 年，休斯的一颗卫星发生事故，卫星的通讯中断了96 秒之后重新恢复。科学家 Edward C.Smith 博士、Al Holman 和 Dan Binder 博士将这种异常解释为单粒子翻转（SEU），并于 1975 年在 IEEE《Nuclear Science》上发表了第一篇 SEU 论文。1979 年，Timothy C.May 和 M.H.Woods 描述了封装材料中 α 粒子造成软错误的第一个证据。1979 年，IBM 的 James Ziegler 和耶鲁大学的 W.Lanford 首次描述了海平面宇宙射线（sea-level cosmic ray）在电子器件中引起单粒子翻转的机制

单粒子效应产生的影响有很多种，可以被分为非破坏性效应以及破坏性效应。单粒子效应中，非破坏性效应包括：（1） 单粒子翻转（SEU，Single Event Upset）；
（2） 单粒子瞬态脉冲（SET，Single Event Transient）；
（3） 单粒子功能中断（SEFI，Single Event Functional Interrupt）。

单粒子效应中，破坏性效应包括：
（1） 单粒子闩锁效应（SEL，Single Event Latch-up）；
（2） 单粒子烧毁（SEB，Single Event Burnout）；
（3） 单粒子栅击穿（SEGR，Single Event Gate Rupture）。

当耗尽区收集到的电荷 Qcoll 大于临界电荷 Qcrit 时，电路就会发生软错误，软错误指的是半导体器件在经过辐照影响之后，内部的状态发生的暂时性的改变， 之所以叫做软错误，是因为器件在断电重启后软错误会消失。当耗尽区收集到的电荷 Qcoll 小于临界电荷 Qcrit 时，器件就不会发生错误。

3 辐照损伤机理

想要研究清楚辐照对微电子器件材料的影响，最主要的就是要搞清楚辐照对材料的损伤原理，从而针对损伤原理下手，改进器件的结构，更换材料类型，对微电子器件进行抗辐照加固。目前来说，对辐照损伤机理的研究主要从以下三个方面展开：位移效应、电离效应、表面效应。

3.1 位移效应

辐射位移效应是指半导体器件材料在受到辐照后，内部一些原子会被辐照粒子轰击，离开原来的位置产生晶格缺陷，这些缺陷会减小少子寿命。辐照所带来的具有很高能量的粒子，会与半导体材料中的原子或分子发生碰撞，能量在这个过程中会发生传递，在晶格中拥有固定位置的原子，会从辐照射入的高能粒子中吸收能量，原子可以离开晶格的固定位置，到达晶格中的缝隙中去，我们称此时这个原子为间隙原子，原子之前固定的位置为空位。

材料晶格在辐照的影响下发生的物理损伤，主要发生在材料内部。高能中子、电子、质子、重的荷电粒子和 γ 射线都可以使半导体材料发生位移效应。从宏观上来看，它们引起的半导体材料的性能改变比较一致，但从微观过程上来看，不同粒子的辐照位移效应原理是不同的。

单个原子的辐照位移，会连同形成的空位共同构成点缺陷。在最开始的时候， 形成的缺陷都是点缺陷。由于温度会发生变化，一些空隙与原子又会重新复合， 使得点缺陷消除。除了会使得缺陷消除，通常情况下，温度的变化会产生两种不同的新的缺陷类型：其一是简单缺陷，它是一种相对稳定的缺陷，一般是由几个原子的位移造成的，如两个空位或三个空位等；其二就是缺陷团簇，它是一种比较大的缺陷，由几百个甚至上千个缺陷组成，缺陷直径可能在 500 埃到 2000 埃不等。晶格缺陷会在晶体中形成新的复合中心，降低少子寿命，同时造成载流子迁移率降低，器件性能退化。

3.2 电离效应

电离效应就是指半导体材料中的中性原子或分子在受到辐照后电离，产生电子空穴对，改变电导率，破坏材料内部的电中性环境。当电离后的电子还具有较高能量时，它还会继续激发其他的中性原子产生新的电子空穴对。在有电场的情况下，新产生的电子空穴对会分别向正极和负极移动，形成光电流，影响器件正常工作。对于金属来说，电离效应不会造成较大影响，因为金属中本来就有大量的自由电子，新电离出来的电子与空穴不会影响金属的电学性质。但对于半导体材料来说，电离出来的新的电子空穴对，则会对材料的物理、化学性能产生较大影响

3.3 表面效应

表面效应就是半导体表面的氧化层中因为辐照而产生的电离效应，陷阱电荷和界面态的产生过程如下:

（1） 电离辐照产生了大量电子和空穴。

（2） 一些电子和空穴立即复合。

（3） 空穴（质子）在电场作用下在氧化层中传输。

（4） 栅极下方的 SiO2 层中会存在很多陷阱，他们会捕获一部分传输的空穴。

（5） 当空穴运动到边界处时，会在 SiO2 一侧形成界面态电荷。

辐照在 Si-SiO2 界面上产生了空穴电子对。在空穴电子对产生后，有一部分会立即复合。自由电子如果还没有和空穴复合，会被电场迅速扫出氧化层，而空穴则不会，因为电子在氧化层中迁移率远大于空穴。没有复合的自由空穴就在电场的作用下，在氧化层中向界面传输。在空穴向界面传输过程中，栅极 SiO2 层中存在陷阱，它们会把一部分空穴捕获形成陷阱电荷。在 SiO2 与 Si 的界面附近，陷阱电荷和界面另一边的 Si 交换电荷，这是因为发生了隧道效应 。空穴到达界面附近后，通过一些较为复杂的反应，形成界面态电荷。
